2206 문제 풀이
 이 문제는 1시간 내에 풀지 못하여 구글링을 통해 문제 해결법을 보고, 2시간 정도 쏟은 뒤에 풀었다.
 문제 풀이의 핵심은 BFS를 통한 그래프 이동 + 그래프 이동시 벽을 뚫었었는지 확인하는 부분으로 총 2가지다.
 그래서 처음에 풀었을 때는 그래프 이동과 값 저장한 배열이 다른 곳에서 알고리즘을 풀어서 틀렸던 것이다.
